# Integration Roadmap

PayXCommerce maintains SDKs and ecommerce connectors for common merchant integration paths.

## Available SDKs

- PHP
- Python
- Node.js

## Available Ecommerce Connectors

- WooCommerce
- OpenCart 3
- OpenCart 4
- Magento 2

## Planned Expansion

Additional SDKs and platform connectors may be added based on merchant integration needs, platform adoption, and support coverage requirements.

Priority candidates include:

- Java
- .NET / C#
- Go
- Joomla / VirtueMart

New connectors should follow the same security model: server-side credentials, hosted checkout redirects, signed webhook verification, idempotent request creation, and redacted logging.
