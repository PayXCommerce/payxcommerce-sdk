# PayXCommerce Magento 2 Payment Module

Developer-preview Magento 2 module for PayXCommerce hosted checkout.

## Package Name

```text
PayXCommerce_Payment
```

## First Release Flow

1. Customer selects PayXCommerce at checkout.
2. Module creates a PayXCommerce payment request.
3. Customer redirects to hosted checkout.
4. PayXCommerce webhook updates Magento order status.

## Scope

- Magento 2 only.
- Magento 1 is not supported.
- Hosted checkout only; no embedded card collection.
